# docact
A Docker container to build documentation for a Python/C++ project using Sphinx Breathe and Exhale


## Selected Python package versions
| Package                      | Version    |
|------------------------------|------------|
| breathe                      | 4.33.1     |
| Sphinx                       | 4.5.0      |

## Usage

Run: `docker run brille/docact`

Build: `docker build -t brille/docact`
